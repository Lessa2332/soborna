 ПИСАНКА ( Pysanka)

Інтерактивна AR-подорож у світ духовних координат України

[Перейти до застосунку](https://lessa2332.github.io/soborna/)**

Відскануйте QR-код на сайті, щоб одразу спробувати AR на своєму смартфоні. Жодних завантажень — просто відкрийте камеру!

 Про проєкт

«Писанка» — це веб-застосунок з доповненою реальністю (AR), створений для популяризації української культурної спадщини серед дітей та молоді. 

Користувачі можуть оживити на своїй долоні автентичні писанки 14 історичних регіонів України, керуючи ними природними жестами рук. Наприкінці подорожі кожен залишає свій слід на спільній інтерактивній карті, символізуючи єднання України.

Проєкт ідеально підходить для використання в музеях, освітніх закладах та на культурних заходах.

 Для музеїв та закладів культури

Цей проєкт створений як готове безкоштовне рішення для інтеграції в експозиції музеїв

Як використовувати:
1. Розмістіть QR-код, що веде на [сайт проєкту](https://lessa2332.github.io/soborna/), біля експонатів.
2. Відвідувачі сканують код і взаємодіють з AR на власних смартфонах.
3. Жодного додаткового обладнання чи налаштувань не потрібно.
